# PageFlip
A Java library of 3D style page flip. It needs OpenGL 2.0!

## Preview

## Installation

#### Gradle

Add it to your build.gradle with:
```gradle
allprojects {
    repositories {
        maven { url "https://jitpack.io" }
    }
}
```
and:

```gradle
dependencies {
    compile 'com.github.eschao:android-PageFlip:v1.0'
}
```
